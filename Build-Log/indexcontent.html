Introduction and System Diagram

i) Introduction

This project is a collaboration of the Humber Parts Crib which will allows students to sign off items for their labs on regular basis. In short, the main purpose of this whole project is to make the Humber Part Cribs more effective to deal with a large number of students. So basically, I will have a camera which I will use in order to scan the barcode strip on the student card and that barcode will get input electronically in the system. I will be using the LED on the PCB to show if the barcode was scanned successfully. Once it is scanned, that barcode information is sent on a txt file where the user can see and have access to it.

Mechanical Assembly

The mechanical assembly for this project is actually very simple. All you have to do is get your raspberry pi 3 connected on the monitor.

Steps:
  • Plug your power supply which is connected to your raspberry pi,

  • Use a VGA to HDMI to your monitor,

  • Connect your keyboard and mouse to your raspberry pi,

  • Don't forget to configure your monitor to VGA settings,

  • Boot up,

  • After that, connect your camera,

  • Go to the appropriate file directory and run the program using the appropriate commandline,

  • Hold your barcode in a fixed and steady position,

  • Watch the LEDs light, once it turns green. You're good to go,

  • Use the command "cat" to concatenate the .txt file and see your information.

    • This is basically the mechanical assembly and the overall steps that a user needs to remember in order to be able to use this hardware equipment.

    PCB/Soldering

    First, I would like to share my PCB soldering task which I completed in week 4. Please find a picture below:

    The PCB consists of a 20 pin GPIO header, two 4 pin header, 5 pin header, resistors, transistors, LED and some jumpers.

    The PCB is also known as the Modular Sensor Hat. I got it from Humber College. The soldering was pretty simple as the schematic was given to us showing where the components are supposed to go. The only challenging thing for me was the soldering because it was my first time. However everything went good and the soldering was successful. While soldering, I had to make sure that I am wearing my safety glasses for my safety and avoid from having any sort of liquid (water, coffee etc..) around me which might be harmful.

    Power Up

    After the mechanical assembly is completed, you can now power up your raspberry pi. Make sure everything is connected. Update your raspberry pi to the latest version if you haven't done so. Use the command "sudo apt-get update. You can either run the traffic2B program by running the command "sudo./traffic2B" or you can run the barcode scanner program by using the command "python programname.py", this will start the window for your camera and your should be able to scan the barcodes.
